# Sweat & Stats

A mobile-friendly personal workout tracker built as an installable Progressive Web App.

**Tagline:** Every Rep Counts.

## Files to upload

Upload every file from this package to the same GitHub repository folder:

- `index.html`
- `manifest.json`
- `service-worker.js`
- `icon-64.png`
- `icon-192.png`
- `icon-512.png`
- `icon-maskable-512.png`
- `apple-touch-icon.png`
- `README.md`

The app stores workout information locally in the browser on the device. Replacing the website files does not normally erase that browser data.
